    Tools¶

    Use the clipboard from the command line

    Related tip and super power tool is using a cliipboard manager (a tool that lets you copy, cut and paste items as many times as you need). In windows it's built in, win+V. On my mac I use the clipboard tool built into "BetterTouchTool", which I've customized in a few different ways. Flycut is another popular option.

    Admit your weaknesses

    Making mistakes is OK One line I take issue with, If you work in an environment where you cannot acknowledge errors, make plans to leave as soon as possible. I completely agree with the sentiment; you should never need to hide mistakes. However, consider a more nuanced approach than immediately finding a new place to work. Try and change the culture if you can.
